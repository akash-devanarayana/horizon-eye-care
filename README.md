# Horizon

A minimal Windows desktop app that follows the **20-20-20 rule**: every 20 minutes,
look at something ~20 feet away for 20 seconds.

Horizon runs quietly in your system tray and gently reminds you to rest your eyes with
a calm full-screen break, then gets out of your way.

## Features

- **Break reminders** — A full-screen overlay with an animated circular countdown.
- **Random break visuals** — Each break shows one of four calming scenes: a sunrise,
  a drifting starfield, gentle rain, or a breathing-guide circle.
- **Skip button** — Dismiss a break early when you're mid-flow.
- **Custom intervals** — Set your work interval and break duration (with quick presets).
- **Daily goal** — Pick a target number of breaks per day and track progress toward it.
- **Sound** — Gentle chimes at the start and end of a break (toggleable).
- **Focus shield** — Automatically pauses during full-screen apps (games, video,
  presentations) so a break never interrupts you.
- **Stats** — Current streak, personal best, today's breaks, all-time total, and a
  7-day bar chart with your goal line.
- **Themes** — Switch between a light **Daylight** look and a dark **Midnight** theme.
- **Start with Windows** — Optionally launch Horizon automatically at sign-in.
- **Update notifications** — Tells you when a new version is available.
- **System tray** — Runs in the background: pause/resume, settings, stats, check for
  updates, and quit.

## Install

Download the latest **Horizon Setup x.y.z.exe** from
[Releases](https://github.com/akash-devanarayana/horizon-eye-care/releases) and run it.

> The installer is not code-signed, so Windows SmartScreen may warn on first run —
> choose **More info → Run anyway**.

## Usage

Everything lives in the system tray icon (bottom-right of the taskbar):

- **Left-hover** shows the time until your next break.
- **Right-click** for the menu: Pause/Resume, Settings, Stats, Check for Updates, Quit.

Open **Settings** to adjust intervals, daily goal, sound, focus shield, auto-start,
and theme. Open **Stats** to see your streak and weekly chart.

## Development

Requires [Node.js](https://nodejs.org) (v18+).

```bash
# Install dependencies
npm install

# Run the app in development
npm start

# Build the Windows installer (output in dist/)
npm run dist
```

## Project structure

```
src/
  main.js          Main process: timer loop, tray, windows, IPC, fullscreen detection
  preload.js       Safe IPC bridge exposed to the windows
  overlay.html     The full-screen break screen (animations + progress ring)
  settings.html    Settings window (editorial UI)
  stats.html       Stats window (streak, cards, 7-day chart)
  update-toast.html  "Update available" notification card
  fonts.css        Bundled Space Grotesk font (embedded)
  theme.css        Daylight / Midnight theme variables
assets/
  icon.ico / icon.png   App + tray icon
  sounds/               Break chimes (generated by generate-sounds.js)
```

## Tech stack

- [Electron](https://www.electronjs.org/) — desktop framework
- [electron-builder](https://www.electron.build/) — Windows NSIS installer packaging
- [koffi](https://koffi.dev/) — calls the Win32 `SHQueryUserNotificationState` API for
  full-screen / do-not-disturb detection
